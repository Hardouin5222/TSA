# Web Foundation

## Amaç

MVP web deneyimi için mobil öncelikli, güven veren ve çapraz satış potansiyeli yüksek bir ana sayfa temeli kuruldu.

## Tasarım Yönü

- Mobil öncelikli yapı
- 3 tık mantığına yakın arama akışı
- Turna benzeri hız hissi
- Enuygun benzeri güven ve profesyonellik
- Odamax benzeri dönüşüm odaklı çapraz satış blokları

## Bu fazda eklenenler

- Next.js app router iskeleti
- temel tasarım sistemi ve global stiller
- ana sayfa hero alanı
- booking vertical sekmeleri
- arama kutusu taslağı
- güven sinyalleri
- çapraz satış kartları
- popüler rota blokları
- deneyim prensipleri bölümü

## Sonraki frontend sprintleri

1. Login ve register sayfalari
2. Gercek auth API entegrasyonu
3. Header state ve kullanici oturumu
4. Arama formunu backend kontratina yaklastirma
5. Son aramalar ve populer destinasyon verisi
6. Tasarim sistemini ortak `packages/ui` altina tasima

## Not

Bu asamada odak, profesyonel bir gorsel ve UX temeli olusturmaktir. Gercek veri akislari bir sonraki sprintte backend auth ve arama modulleriyle baglanacaktir.
